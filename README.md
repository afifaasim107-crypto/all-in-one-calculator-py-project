# All In One Calculator Python Project

##  Project Overview
This is a multi-functional Python project that includes different types of calculators in one repository.  
The purpose of this project is to practice Python fundamentals and build real-world logic-based programs.

## Features
- Business Investment Calculator
- Financial Calculator
- Conversion Measurement Calculator
- Health & Wellness Calculator
- Utility Calculator

## Technologies Used
- Python
- Basic Programming Concepts
- Conditional Statements
- Functions

## Learning Objectives
- Improve problem-solving skills
- Understand user input handling
- Practice mathematical logic in Python
